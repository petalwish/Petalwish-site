# 交互逻辑
## DNA 引导的生命历程
当访客进入页面，视角是从 DNA 的微观视角切入的。随着滚动：
解旋与重组： DNA 随着滚动进度不断旋转、延伸。
锚点映射： 螺旋上的每一个“碱基对”或特定节点，都是一个内容触发点。当某个节点滚动到屏幕中心，侧边会浮现出你的项目（如 GenomicAI Engine）或研究方向。
材质演变： 初始可能是有机的生物质感，随着向下进入“技术区”，螺旋逐渐碎片化，演变为由 0 和 1 或者代码片段组成的几何阵列。

## 针对 Markdown 博客的“阅读绿洲”策略
你担心深色影响阅读，但在“无限画布”中，我们可以利用空间转换来解决：

沉浸式阅读区： 当用户点击进入某篇博客时，我们可以通过一个丝滑的过渡动画，让 DNA 螺旋向左/右侧退场，页面中心升起一块半透明的、具有磨砂质感的“电子纸”容器。

对比度控制： 正如之前提到的，背景使用深蓝黑（如 #0F172A），文字使用暖灰色（如 #CBD5E1）。这种配比在保证“代码编辑器”酷感的同时，能让视觉疲劳度降到最低。

代码高亮： 既然是 Markdown，我强烈建议使用 Shiki 配合 Tokyo Night 或 Github Dark 主题，这会让你的代码块看起来非常专业。

# 交互方案
## Section 1: Hero Section (MANIFESTO)
视觉： 整个屏幕只有极深的底色（Near-black）。在屏幕中心，一个极细、发着微弱蓝光的 3D 点云 DNA 双螺旋在静静地自转。
文字： Code is the new pipette. 这句话使用 JetBrains Mono 字体，像打字机一样一个字母一个字母地“敲”在屏幕上。
交互： 鼠标悬停在 DNA 上时，点云会像被磁铁吸引一样轻轻扰动。


## Section 2: 核心展厅 (The Lab Clusters)
随着你的向下滚动，场景开始变换：
药学生/X-Lab： DNA 螺旋会放大，视角进入螺旋内部。你会看到碱基对像巨大的支柱一样掠过。侧边浮现出你的履历块。
Surfing AI/未来方向： 随着滚动，原本平滑的 DNA 结构开始出现**“数字畸变”**。它不再是单纯的生物模型，而是由数千个飞舞的代码片段、公式（如 $\partial L/\partial \W$）和分子式凝聚而成的、不规则但有序的形状。
交互细节： 每当你滚动到一个新的阶段，背景中会隐约传出键盘敲击声或实验室仪器发出的微弱蜂鸣声（极短的 3D 音效），增强沉浸感。


## Section 3: Blog (思考模型)
这是你担心深色模式影响阅读的地方。
过渡： 当滚动手势进入博客区域，主体的 3D DNA 螺旋会优雅地解旋，平铺开来，在屏幕边缘形成两条由数据流组成的装饰性线条。
阅读容器： 屏幕中央升起一个全屏宽度的、极低透明度的 frosted glass（毛玻璃）容器。它将背景的 3D 动效模糊化，只保留光影感。
排版： 正文使用 Inter 字体，增大字间距和行高（line-height: 1.8），确保长时间阅读不累。你可以用这块区域展示你关于 Agent 和精益创业的分析文章。


## Section 4: 履历 & 联系页 (The Outro)
视觉： 这是一个视角抬升的过程。从微观的 DNA 再次拉远，看到整个完整的螺旋。在螺旋的最下方，是高中时期，向上依次是大学、AI 阶段。
联系方式： 最后的二维码不要只是呆板的白底黑字。我们可以给二维码做一个霓虹发光（Neon Glow）的描边，或者点击它时，二维码像细胞分裂一样炸开，变成几个社交媒体的 Logo。
